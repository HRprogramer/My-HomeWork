Lists

To make lists, we use the following tags:

  1. OL tag : To meke a list in terms of numbers,we use the (OL) tag and write each line of the list in the (LI) tag.

  2. UL tag : An unordered list starts with the (UL) tag and each line is written in the (LI) tag.

Tables

  • We use the (Table) tag to make the table.
  • We make the table row by row.
  • We use the (TR) tag to start building a row.
  • We use the (TD) tag to build each house from the table columns.
  • So first the (Table) tag , than the (TR) tag and then the (TD) tag.
