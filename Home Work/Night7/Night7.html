In this article, we are looking to learn more about family properties and how to use them.



Styling Text

  1. Font-family

    2. We can change the font of the text using the font-family property.
    Note: If the font name consists of multiple words, it has to be in double Quotation (" ").
    For example:

    .test{

    font-family : Arial ;

    }

     Note : Some browsers do not support a series of fonts, so if you want to use a font for your site, add other fonts.
    for example:

    .test {

    font-family : Arial , Helvetica , sans-serif ;

    }

    Note : Fonts that are in all systems(These are called web safe fonts) : Arial, Courier New, Georgia, Times New Roman, Trebuchet MS, Verdana .

  2. Font-style

    3. The font style property is used to make text italic.

    .test {

    font-style : italic ;

    }

  3. Font-weight

    4. The font-weight property is used to make the text bold.

    .test {

    font-weight : bold ;

    }

    Note: We can define the font weight with a number from 100 (thin) to 900 (thick). 400 is the same as normal , and 700 is the same as bold .

Font Styles

  1. text transform

    2. With the help of this property, we can convert the text to the following .
    1. uppercase: Transforms the text to capitals.
    2. lowercase: Transforms the text to lower case.
    3. capitalize: Transforms all words to have the first letter capitalized.
    For example:
    .test1 {
    text-transform : uppercase ;
    }

    .test2 {
    text-transform : capitalize ;
    }
    Output:
    (test1)I am a programmer!

    (test2)I am a programmer!

  2. text decoration

    3. The text-decoration property supports the following values:

    1. none: removes any decoration. -----» I am a programmer
    2. overline : gives the text an overline. -----» I am a programmer
    3. underline: underlines the text. -----» I am a programmer
    4. line-through: puts a strikethrough over the text.-----» I am a programmer

  3. text shadow

    4. The text-shadow property allows you to set a shadow for the text.
    This property requires four values:

    1. The horizontal offset of the shadow.
    2. The vertical offset.
    3. The blur radius: a higher value means the shadow is dispersed more widely.
    4. The color of the shadow.

    For example:
    p {
    text-shadow: 3px 2px 3px red;
    }

    Output:

    This is a paragraph!

    Note:We can add multiple shadows, by separating them with a comma( , ) .
    p {
    text-shadow: 3px 2px 3px red , 2px 2px 1px blue;
    }

  4. text-align

    5. The text-align property is used to control how text is aligned within its container.
    The values ​​that this property takes include: { left , right , center , justify }

    For example:
    .test{

    text-align : center ;

    }

  5. line-height

    6. line-height property Determines the distance between lines and its input value is a number.
    For example :
    .test{

    line-height : 1.8 ;

    }

  6. word-spacing & letter-spacing

    7. They allow us to adjust the spacing between words or letters in our text .
    For example:

    .test {

    letter-spacing: 3px;
    word-spacing: 3px;

    }

The box model

HTML elements appear as boxes.
The box model defines how the different parts of the box.
margin , border , padding , and content , work together to create the box.

1. border : It is the walls of the box.
2. margin : It is the same distance between the content section and the walls of the box.
3. padding : The distance between the walls of the box and other elements is called margin.

Further getting to know

  • weight

    • Determines the width of the box.
    .test {
    weight : 200px ;
    }

  • height

    • Determines the length of the box.
    .test {
    weight : 200 ;
    height : 100 ;
    }

  • border

    • We define the walls of the box with it.
    Note : Borders, like paddings and margins, have top & bottom & left and right sections.

    This property requires three values to work:
    1. Border thickness : (number)
    2.The border itself : The border can be continuous (solid) or interrupted (dashed)
    3.color
    For example:
    .test {

    weight : 200px ;
    height : 100px ;
    border : 3px solid red ;
    }

    Output
    Note the border.
    Note : The border will be invisible if its style is not defined. This is because the style defaults to none.

  • Box

    • All the contents of the site are inside the boxes that we can style these boxes with the help of CSS.
    Two important terms:

    1. box-sizing
    The size of the box is determined in two ways:

    content-box : gives you the default CSS box-sizing behavior.
    If you set an element's width to 100 pixels, then the element's content box will be 100 pixels wide,
    and the width of any border or padding will be added to the final rendered width, making the element wider than 100px.

    border-box : tells the browser to account for any border and padding in the values you specify for an element's width and height.
    If you set an element's width to 100 pixels, that 100 pixels will include any border or padding you added, and the content box will shrink to absorb that extra width.
    *This typically makes it much easier to size elements.*

    .test {
    height: 200px;
    width: 500px;
    box-sizing: border-box;
    }

    2. box-shadow

    We apply the shadow we give to the box using the box-shadow property It needs several values:
    1. The length of the shadow on the x-axis
    2. The length of the shadow on the y-axis
    3. The value of the shadow clarity
    4. The color of the shadow

  • Padding

    • The distance between the walls of the box and the content is called padding.
    Padding should have four values:

    1. padding-top
    2. padding-right
    3. padding-bottom
    4. padding-left

    Paddings can be set in the following ways:

    /* padding : padding-top | padding-right | padding-bottom | padding-left ; */
    padding: 5px 10px 15px 20px;

    /* padding : padding-top | padding-right & padding-left | padding-bottom ; */
    padding: 5px 10px 20px;

    /*padding : padding-top & padding-bottom | padding-right & padding-left ; */
    padding: 5px 20px;

    /*padding : padding-top & padding-bottom & padding-right & padding-left ; */
    padding: 20px;

  • Margin

    • The margin is not counted towards the actual size of the box, it is the space outside the box
    The margin is the space around your box.

    1. margin-top
    2. margin-right
    3. margin-bottom
    4. margin-left

    margins can be set in the following ways:

    /* margin : margin-top | margin-right | margin-bottom | margin-left ; */
    padding: 5px 10px 15px 20px;

    /* margin : margin-top | margin-right & margin-left | margin-bottom ; */
    margin: 5px 10px 20px;

    /*margin : margin-top & margin-bottom | margin-right & margin-left ; */
    margin: 5px 20px;

    /*margin : margin-top & margin-bottom & margin-right & margin-left ; */
    margin: 20px;
