Table

Let's go with an example:

table {
border-collapse: collapse;
width: 100%;
}

td,th {
border:1px solid black;
text-align: center;
}

output :

Mathematics physics chemistry
20 15 18

Note :By default, there is a space between table cells that creates a double border.
using the border-collapse property , we can remove this space.

Let's give space to the contents of the first table for convenience.

td,th {
border:1px solid white;
padding : 10px;
text-align: center;
}

We have also aligned the text in all cells to the center.
Now , it's time to create a background!

th {

background-color: gray;
}

Note :CSS provides the nth-child() pseudo selector, which allows you to select specific rows and style them.

For example, we can style the 3rd row like this:

tr:nth-child(3){

background-color: #b0d1e4;
}

The rows are counted from the top to the bottom, including the heading rows.
The nth-child selector allows you to also define rules for the rows, instead of providing the number.

tr:nth-child (even){

background-color: #b0d1e4;
}

tr:nth-child (odd){

background-color: #6aabd1;
}

Note :This will automatically apply the assigned styles to the odd and even rows.
