Datalist

To create a drop-down list, we use the following tags:

[input id="car" type="text" list="color" placeholder="Choose the color of the car">
[datalist id="color">

[option value="red">
[option value="gray">
[option value="pink">

[/datalist>

For example:

**The list attribute value in the input tag should be equal to the id value in the datalist tag.
