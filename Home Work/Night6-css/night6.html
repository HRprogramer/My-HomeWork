Css & Html

We use css language to create the appearance features of our site.
Appearance features such as text style, font size, changing color, etc.


How to write Css codes ==> Property: value ;
For example:

font-size : 28px ;

Styling

CSS allows you to target elements using selectors,which allows you to target specific elements and apply the same style to them.
There are several ways to do this :

  1. (Direct) Attribute style :

    2. This method is done directly while writing Html codes.
    By putting style Attribute in target tag, we can use the services of this Css code.
    For example: Style="color:red; font-size: 28px;"

    Note:The declarations are separated using a semicolon.

  2. (Indirect) Selector:

    3. Note:To use this method, you must add the link tag in the head tag of the site.

    p {
    color:red;
    }

    The selector p targets all the paragraph elements of our HTML.
    This CSS command is applied to all the p tags on the codes.
    So this method is not very effective.

  3. (Indirect) Multiple selectors

    4. We can target Multiple selectors at once, by separating the selectors with a comma.
    For example , let's make all h1 heading and Paragraphs red :

    h1 , p {
    color : red;
    }

    This a paragraph!!

  4. (Indirect) Create class :

    5. In the explorer part of the program, we create a page with CSS extension.
    We first need to add a class attribute to the elements we want to target :

    «p class="intro"» CSS is awesome «/p» *

    Now, we can create the class selector in CSS And in the CSS page , enter the following codes:

    .intro {

    color : red ;
    backround-color : yellow ;

    }

    *I used «» instead of <> .

  5. (Indirect) Create ID :

    6. In the explorer part of the program, we create a page with css extension.
    The id selector method is the same as the class selector, except that we use (#) instead of a (dot) in the CSS page.
    for example:

    create class :
    * .intro

    create ID:
    * #intro

    Note:We can use the classes in other places, but we can only use the IDs in one place, so the names of the IDs must be unique.

  6. (Indirect) Addressing by location :

    7. We can also target elements based on their location.
    for example, there are two «strong» elements, one inside a paragraph and the other inside a list item.
    We can target only the one inside the list item by using the following rule:

    li strong{

    color : purple ;

    }
    The space denotes that the «strong» element needs to be inside an «li» element.

  7. (Indirect) Use of attributes :

    8. In this method, by using the place of attributes in the program, we give them the desired style.
    We do this in the following ways:

    «input type ="text"»

    and in the CSS page :

    [type="text"] {
    border: 2px solid green;
    }

    Note: In this method, we bring attribute and select them using [ ].
