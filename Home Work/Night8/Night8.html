background

Let's get to know this property better:

  1. Background-Color

    2. With the help of this property, we can determine the background color of anything.

    .box {

    background-color: red;
    height:100px;
    padding: 10px;

    }

    Note : You can set the color using color names , hex values and rgb.

  2. Background-Image

    3. This property helps us to choose an image as background.
    It has the following syntax:

    .box{

    background-image: url ('MyImage.png');
    height:150px;
    padding: 10px;

    }

    We put the image address in the url() attribute.
    By default, a thumbnail is tiled to fill the frame.
    If the image is larger than the frame, it will not shrink by default , We will only see the part that fits in the box.

  3. Background-Repeat

    4. This property can determine how to repeat or tile the background in the box.
    It takes value in four ways:

    1. no-repeat: does not tile the image.
    2. repeat-x: repeat horizontally.
    3. repeat-y: repeat vertically.
    4. repeat : repeat in both directions (default value).

    .box {

    background-image: url ('MyImage.png') ;
    background-repeat: repeat ;
    height:150px ;
    padding: 10px ;

    }

  4. Background-Position

    5. The background-position property sets the position in which the background image appears.
    It uses a coordinate system in which the top-left corner of the box is at position (0,0).

    Note : This property can take either pixel / percent values or keywords (top, bottom, left, right, and center).

    .box {

    background-image: url ('MyImage.png') ;
    background-repeat: no repeat ;
    background-position: center center ;
    height: 200px ;
    padding: 115px ;
    border: 2px solid red ;

    }

    Note :The default background-position value is (0,0): the top left corner.

  5. Background-Size

    6. The background-size property allows we to define how the image should fit inside the box.
    It takes value in four ways:

    1.Cover : The image will completely cover the box, while maintaining its aspect ratio.

    2.contain : The image will fit inside the box completely.

    For example:

    .box {

    background-image: url ('MyImage.png');
    background-repeat: no repeat;
    background-size: contain;
    height: 200px;
    padding: 15px;
    border: 2px solid red;

    }

Styling Lists

By default, unordered list items are marked with round bullets, while ordered lists are numbered.

Let's get to know this property better:

  1. list-style-type

    2. It's time to change the marker!
    With the help of this property, we can remove markers from the list or change them.
    For example:

    ul {

    list-style-type: dicc;
    }

    Output :

    • Telegram
    • Instagram
    • Twitter
    • Linkedin
    • GitHub


    Note :The list-style-type property allows you to change the markers to circle , square , decimal , disc , lower-alpha , lower-roman , etc .

  2. list-style-position

    3. This property determines whether the position of the markers is outside or inside.
    Pay attention to the examples:

    ul{

    list-style-position: inside;
    }

    Output:

    • Apollo 11: First Human Landing
    • City in Space: The International Space Station
    • Great Observatory: The Hubble Space Telescope
    • Everlasting Mars Rovers
    Note :As you can see, the bullets are inside the text.
    To better understand this, compare this example above with the next example.

    ul{

    list-style-position: outside;
    }
    Output:

    • Apollo 11: First Human Landing
    • City in Space: The International Space Station
    • Great Observatory: The Hubble Space Telescope
    • Everlasting Mars Rovers
    Note :The default value is outside.

  3. list-style-image

    4. Sometimes we say that we wish we used an image instead of markers in our list.
    The list-style-image property is here to help us !

    ul {

    list-style-image: url ("bullet.jpg") ;
    }

    Note :This property does not allow us to control the size of the image, so:
    1. it's better to set the list-style-type to none
    2. and add a background image to the list items with the marker image,
    3. positioning it to the left.

  4. list-style (One time for ever)

    5. The list-style property is a shorthand for setting list-style-type , list-style-image and list-style-position , all in one declaration.

    ul {

    list-style: square outside none;
    }

    This would be the same as:

    ul {

    list-style type: square;
    list-style position: outside;
    list-style image: none;
    }

Links

A link is by default blue in color and underlined.
We can style links with CSS properties (color , font-family , background , etc).

For example:

a {

font-size : 28px ;
}

In addition, links can be styled differently, depending on what state they are in.
The following selectors are available :

a:link --» defines the style for normal unvisited links.
a:visited --» defines the style for visited links.
a:active --» a link becomes active once you click on it.
a:hover --» a link is hovered when the mouse is over it.

Note :These are called pseudo selectors.
